##  Sobre o Projeto
 
O **Fjoren** é um sistema web de comércio eletrônico desenvolvido para a venda de roupas com integração de um **provador virtual**, permitindo que os usuários visualizem como as peças ficariam antes de realizar a compra.
 
Atualmente, muitas lojas virtuais apresentam dificuldades para os clientes visualizarem como as roupas realmente ficarão no corpo. O Fjoren resolve esse problema ao oferecer uma solução tecnológica que aproxima a experiência da compra online da experiência de uma loja física — reduzindo dúvidas sobre tamanho, estilo e ajuste das peças.
 
O sistema permite que **clientes** naveguem, experimentem virtualmente e comprem roupas, enquanto **administradores** gerenciam produtos, estoque, pedidos e comunicação com os clientes de forma estruturada e intuitiva.
